# MMEC Study Archives

A centralized academic resource platform for college students to access syllabi, notes, and previous year question papers in one place.

## 🚀 Features

* Organized academic resources
* Mobile responsive design
* Fast static website
* Easy navigation
* Zero-maintenance deployment

---

## 🛠️ Tech Stack

* HTML ,CSS ,JS
* Netlify

---


## ⚙️ Installation

```bash
git clone https://github.com/YOUR_USERNAME/mmec-study-archives.git

cd mmec-study-archives

npm install
```

---

## ▶️ Run Locally

```bash
npm run dev
```

---

## 🌐 Live Demo

https://mmec-study-archives.netlify.app

---

## 📚 Motivation

Students often waste time searching through WhatsApp groups and random drives for study material. This platform centralizes everything in one place.
